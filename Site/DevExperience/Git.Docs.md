

# Git

## Passos básico para push de ficheiros

    git init
    git add README.md
    git commit -m "first commit"
    git branch -M main
    git remote add origin «Url Repositorio»
    git push -u origin main

## Listar todas as origens

    git remote -v

## Clonar um repositorio 

    git clone «url do repositorio»
